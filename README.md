## 使用方法
1. 直接執行主程式  
2. 程式啟動後會自動開啟一個網站畫面  

---

## 網站功能說明

### 一、查詢系統
- 使用者可依照：
  - 欄位條件查詢
  - 測站名稱查詢
- 點擊 **Detail** 按鈕後：
  - 顯示該筆資料的詳細資訊
- 頁面下方：
  - 顯示資料庫中的全部資料

---

### 二、圖表系統
- 使用者可先選擇欲觀察的測站
- 選擇完成後：
  - 下方即顯示該測站之 **PM2.5 長條圖**

---

### 三、帳號系統
- 提供使用者：
  - 帳號註冊功能
  - 登入系統功能

---

## 資料庫建立
- 系統會將取得的 AQI JSON 資料存入資料庫
- 提供後續查詢與圖表分析使用
